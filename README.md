# Restaurant Point of Sales (POS) System

## Overview
A front-end Point of Sales web application built to handle restaurant orders. This project demonstrates DOM manipulation, event handling, and dynamic UI updates using standard web technologies.

## Features
* **Dynamic Menu:** Menu items and prices are rendered dynamically using JavaScript.
* **Shopping Cart:** Users can add items, specify quantities, and see their running total update in real-time.
* **Checkout Logic:** Includes payment validation to check for insufficient funds and automatically calculates the customer's change.

## Technologies Used
* HTML5
* CSS3 (Bootstrap 5 for layout)
* Vanilla JavaScript (ES6)

## How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No local server or build tools are required.
